# [Plugins](https://claude.ai/docs/code/plugins)

Plugins let you extend Claude Code with custom functionality that can be shared across projects and teams.
Plugins let you bundle all your customizations into shareable packages that install with a single command.

https://claude.com/blog/claude-code-plugins

## [Discover and install prebuilt plugins through marketplaces](https://code.claude.com/docs/en/discover-plugins)

A marketplace is a catalog of plugins that someone else has created and shared.
The official Anthropic marketplace (`claude-plugins-official`) is automatically available when you start Claude Code.

https://github.com/anthropics/claude-code/tree/main

## [Claude Code Templates](https://www.aitmpl.com)

Templates to enhance Claude Code's capabilities for specific tasks or domains.
